PCA_Classification.pca = PCA(n_components=50, random_state=75)¶

INBUILT FUNCTION : PCA(n_components=50, random_state=seed)

Extended description of function

PCA is a dimensionality-reduction method that is used to reduce the dimensionality of large data sets, by transforming a large set of variables into a smaller one that still contains most of the information in the large set.

Parameters

n_components : (int) Number of components to keep. If n_components is not set all components are keep. It representing the directions of maximum variance in the data. The components are sorted by explained_variance.

random_state : (int) It has determines random number generation for subsampling and smothing noise

INBUILT FUNCTION : QuantileTransformer(n_quantiles=10, random_state=seed)

Extended description of function

This method transforms the features to follow a uniform or a normal distribution. For a given feature, this transformation tends to spread out the most frequent values.

n_quantiles : (int) The actual number of quantiles used to discretize the cumulative distribution function. random_state : (int) It has determines random number generation for subsampling and smothing noise.
