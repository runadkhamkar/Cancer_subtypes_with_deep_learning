SelectBest_Classification.qt = QuantileTransformer(n_quantiles=10, random_state=75)¶

INBUILT FUNCTION : QuantileTransformer(n_quantiles=10, random_state=seed)

Extended description of function

This method transforms the features to follow a uniform or a normal distribution. For a given feature, this transformation tends to spread out the most frequent values.

n_quantiles : (int) The actual number of quantiles used to discretize the cumulative distribution function. random_state : (int) It has determines random number generation for subsampling and smothing noise.
